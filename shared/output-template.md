# Suggested Response Template

Use this compact structure when reporting progress on a challenge.

## Triage summary
- Category guess
- Key artifacts
- Most promising primitive

## Evidence
- Fact 1
- Fact 2
- Fact 3

## Next attacks
1. Best next step
2. Backup path if that fails
3. Required tools or scripts

## Flag handling
Once the flag is recovered, show the exact extraction path briefly and verify formatting before returning it.
